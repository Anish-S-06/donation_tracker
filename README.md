# Donation Tracker

This is the main repository for the Donation Tracker group project.

## Tech Stack
- **Backend:** Python 3.x, Flask, SQLAlchemy, MySQL
- **Frontend:** HTML5, CSS3, Bootstrap 5, Jinja2
- **Tools:** Flask-Login, Flask-Mail, Flask-Migrate, bcrypt, Google OAuth, Twilio

## Project Modules
- Database & Backend Core
- Auth & Admin
- Search, Maps & Request System
- Frontend, Notifications & Gamification

# to run
run.py          -> Normal local development
